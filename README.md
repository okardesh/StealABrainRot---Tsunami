# StealABrainRot - Tsunami (Web Edition)

This is a modern, web-based prototype of your game! It's designed to be stunning, safe, and easy to play.

## 🕹️ Controls
- **Movement**: Use **WASD** or **Arrow Keys**.
- **Collect**: Walk over glowing orbs (Brainrots).
- **Deliver**: Bring them to the **Pink Neon Square** in the middle for Gold.
- **Tsunami**: Watch for the **RED WARNING**. A giant blue wave will sweep across. If it hits you, you lose your current item and respawn!

## 💎 Rarities
- **Common** (Gray)
- **Rare** (Blue)
- **Legendary** (Gold)
- **Mythic** (Red)
- **Brainrot God** (Purple)
- **Celestial** (White) - *Super Rare!*

## 🛠️ How to Edit
Right-click `index.html` and choose "Open With" -> "TextEdit" or any code editor.
To change the Tsunami speed, search for `tsunami.speed` in the code.
To change the Gold values, search for `RARITIES`.

Enjoy!
